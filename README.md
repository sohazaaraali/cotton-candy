# cotton-candy
Soha's Sugar Swirls Cotton Candy

## Link to deployed website

https://cotton-candy-jet.vercel.app/

## Pages

- **Home** (`index.html`) - Main landing page with hero section, features, pricing, testimonials, and gallery
- **Flavors** (`flavors.html`) - Browse all available cotton candy flavors
- **Order** (`order.html`) - Place a custom cotton candy order
- **Quote** (`quote.html`) - Request a quote for events or bulk orders
- **About** (`about.html`) - Learn about Sugar Swirls by Soha
- **Business Card** (`business-card.html`) - Printable business cards with QR code

## Printable Business Cards

Visit [business-card.html](https://cotton-candy-jet.vercel.app/business-card.html) to print custom business cards with:
- Business name and contact info
- QR code linking to website
- Print-ready layout (6 cards per page)
